# Evelyn PRO — Carreira & Estudos
- Flask + Google Gemini + PyMuPDF
- Busca vagas (DuckDuckGo) + compatibilidade
- Modo estudos (Método Paulo Freire) via tema ou PDF

python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python backend/app.py
# http://127.0.0.1:5000
